# VisionFest Landing Page

Landing page oficial do VisionFest construída com Next.js, TypeScript e Tailwind CSS.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- ESLint
- Prettier
- Husky
- Vitest
- Playwright
- GitHub Actions

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
npm run typecheck
npm run test
npm run test:e2e
npm run format
```
